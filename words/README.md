This folder contains a list of free English dictionaries.

words - this is the words dictionary from the OpenSuse Tumbleweed package (words-2020.12.07-2.1.noarch.rpm) listed as Public Domain. http://wordlist.aspell.net/

wordtest.txt is a short list of words and non-words used to test this program. It is covered by the same GPL3+ as the program.

wlsolver_dictionary.txt is a combination of CROSSWD.TXT and CRSWD-D.TXT, augmented by new, missing words. It is Public Domain.

All other files in the words folder are from the Project Gutenberg Moby Word II project by Grady Ward. They are Public Domain with information about the project, each file, and the license in the 3201.txt file.
